static的作用
Created2023-10-14
1234static bool cmp(string a, string b){ return a + b < b + a;} 为什么这个函数上面要加static 这个函数上面加 static 是为了限定其作用域，使其只在当前文件中生效，避免与其他文件中的同名函数产生冲突。 在 C++ 中，static 关键字可以用于函数和变量上。对于函数来说，static 的作用是将函数的作用域限定在当前文件中，使其只在当前文件中可见。也就是说，其他文件中的同名函数不会与其产生冲突，避免了命名空间的问题。 在这个例子中，static 关键字将 cmp 函数的作用域限定在当前文件中，避免了与其他文件中的同名函数产生冲突。这在大型项目中尤为重要，因为大型项目中通常包含很多文件，如果不限定函数的作用域，同名函数可能会在不同文件中产生冲突，导致编译错误。 除了限定函数作用域之外，static 还可以用于变量上，用于将变量的作用域限定在当前文件中。
